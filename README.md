# Parametric CAD AI Training Benchmark

This repository contains parametric CAD models, scripted CAD generation examples, and evaluation prompts designed for AI-training workflows in mechanical design.

The goal is to demonstrate how CAD tasks can be structured for model training, evaluation, and quality control using tools such as OpenSCAD, FreeCAD, Python, and exportable STL/STEP workflows.

## Tools Used

- OpenSCAD
- FreeCAD
- Python
- Parametric CAD modeling
- STL/STEP export workflows
- AI prompt/evaluation design
